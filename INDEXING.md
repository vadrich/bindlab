# Индексация BindLab — ~500 SEO-страниц CS2

Каталог: https://bindlab.ru/guides  
Sitemap: https://bindlab.ru/sitemap.xml  

Страницы делятся на:
- `/how-to/*`, `/binds/*`, `/guides/*` — бинды и конфиг
- `/cs2/*` — карты, оружие, ранги, экономика, советы по CS2/ксго2 с CTA в BindLab

Обновить ядро: правь `scripts/generate-seo-catalog.mjs` → `npm run seo:catalog` → `npm run deploy`.

## Google / Яндекс

1. Отправь sitemap заново (стало ~500 URL)
2. Запроси индексирование `/guides` и 10–20 сильных страниц
3. Остальное подтянется из sitemap (может занять дни/недели)

## Замечание

Массовые SEO-страницы дают охват, но Яндекс/Google режут тонкий дубль-контент. Сильнее работают уникальные how-to + внешние ссылки + поведение. При желании позже усилим тексты точечно по топ-запросам из твоего Wordstat.
